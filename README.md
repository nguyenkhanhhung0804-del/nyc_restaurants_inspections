# NYC Restaurant Inspections Analysis

This project analyzes New York City restaurant inspection data using Python and Jupyter Notebook. The objective is to clean and explore inspection records, identify food safety trends, and visualize key insights through exploratory data analysis (EDA).

## Features

- Data collection and preprocessing
- Data cleaning and validation
- Exploratory Data Analysis (EDA)
- Inspection score analysis
- Violation and cuisine trend analysis
- Data visualization
- Statistical summaries

## Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights

- Explored inspection score distributions across restaurants.
- Identified common health code violations.
- Compared inspection outcomes by cuisine type and borough.
- Visualized trends using charts and summary statistics.
